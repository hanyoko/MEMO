1. 메인화면
2. 도서 상세보기
3. 도서 등록하기
- 메인배너, 신간도서목록등이 포함되어 있는 메인화면을 그래픽 소프트웨어를 활용하여 작성하고 메인이라는 이름으로 저장하시오.
- 신간도서 등록하기 페이지 화면을 그래픽 소프트웨어를 활용하여 작성하고 도서등록이라는 이름으로 저장하시오.
- 도서 상세보기 페이지 화면을 그래픽 소프트웨어를 활용하여 작성하고 상세보기라는 이름으로 저장하시오.
- 상단영역과 하단영역은 세 개의 페이지 모두 같은 레이아웃과 형태를 적용하시오.
- 통일성과 사용자의 편의를 고려하여 화면을 설계한다.

그린북스 웹사이트 프로젝트 UI 구현하기
* 화면구현
1. 메인화면은 index.html로 이름을 지정하여 설계한 화면과 동일하기 구현한다.
2. 도서 상세보기 화면은 view.html로 이름을 지정하여 설계한 화면과 동일하게 구현한다.
3. 도서 등록하기 화면은 write.html로 이름을 지정하여 설계한 화면과 동일하게 구현한다.
4. 화면 구현 시 웹 표준 지침과 웹 접근성을 준수한다.
5. 상단영역과 하단영역은 header.html / footer.html로 생성하여 각 화면마다 include 하여 불러와서 사용하도록 한다.
6. css / images / js / include 폴더를 생성 후 각각의 파일을 정리한다.
7. 폼태그 작성 시 폼 태그 속성을 작성하고, 인풋의 타입과 속성을 적절하게 구현한다.
